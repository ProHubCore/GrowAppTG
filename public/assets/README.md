# Игровые изображения

Имена файлов и папок используются кодом через `src/core/assets/assetCatalog.js`.

- Фоны локаций: `backgrounds/`
- Отдельные здания карты: `buildings/`
- Персонажи: `characters/`
- Ведра и реакторы: `containers/`
- Ассеты конкретной комнаты: `locations/<location>/`
- Три стадии каждой культуры: `plants/<slug>/stage-1.png ... stage-3.png`
- UI и инструменты: `ui/`, `tools/`, `seed-packets/`

При простой перерисовке ассета сохраняй существующее имя файла. При добавлении нового ассета зарегистрируй путь в `assetCatalog.js`.
