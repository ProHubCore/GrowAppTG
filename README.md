# РЕЖИМ — MVP 0.3 Clean

Персональная Telegram Mini App для перехода от намерения к действию.

## Что исправлено

- Новый чистый светлый интерфейс без кислотного неона.
- Новый ключ хранения: старые демонстрационные уровни больше не подтягиваются.
- Первый запуск всегда начинается с 1 уровня, 0 XP и пустой истории.
- Сброс удаляет данные всех предыдущих версий.
- Научные механики сохранены, но встроены в сценарий без визуальной перегрузки.
- Production-сборка проверена командой `npm run build`.

## Запуск

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
```

Готовая сборка находится в папке `dist`.
