# Пересборка проекта

## Что изменено

- Все активные экраны собраны в единую feature-структуру без дублирующих старых папок.
- Дядя Джо заменён на Марию Ивановну в обучении, доме, заданиях, доверии, смесях и окнах наград.
- Добавлены отдельные ассеты дома Марии Ивановны, доски дел, радио и персонажа.
- Все изображения переименованы латиницей и разложены по понятным папкам.
- Убраны нарисованные шахматные фоны у отдельных PNG; внутренняя мусорная область под креслом Марии Ивановны очищена.
- Для всех актуальных культур подключены три реальные PNG-стадии.
- Культуры сведены в один каталог `crops.js`, который автоматически питает посадку, магазин, инвентарь, клуб и энциклопедию.
- Добавлена миграция старых сохранений и старых идентификаторов.
- Тяжёлые редкие ассеты больше не блокируют первый запуск: стартовые файлы загружаются сразу, ближайшие локации прогреваются в фоне, остальные изображения загружаются по требованию.
- Добавлена команда `npm run check`, которая проверяет ESLint, все пути ассетов, уникальность культур и production-сборку.

## Проверка
