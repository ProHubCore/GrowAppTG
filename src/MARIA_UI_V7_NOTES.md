# Maria UI v7

- Заголовок доски исправлен на «Поручения Марии Ивановны».
- Убрана лишняя подпись «Доска Марии Ивановны».
- Главы оформлены как неровные деревянные таблички, стрелки сделаны тише.
- Текст главы вертикально центрирован и не касается рамки.
- Незавершённые поручения показываются сверху, завершённые вынесены вниз.
- Карточка выбранного поручения стала компактнее, награда читается сразу.
- Награда главы сохранена, но занимает меньше места.
- Верхняя плашка доверия переработана в кликабельную наградную карточку.
- Каждый уровень доверия можно открыть и посмотреть подробное описание награды.
